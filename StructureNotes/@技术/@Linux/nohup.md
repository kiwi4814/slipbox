# nohup
> NAME
>        nohup - run a command immune to hangups, with output to a [[理解 Linux 中的 tty、pty、pts、console、terminal-FreeOA |non-tty]]
> DESCRIPTION
>       Run COMMAND, ignoring hangup signals.

运行一个不受挂断影响的命令，不输出结果到任何终端。即使退出当前会话和终端，也不会影响程序执行。
